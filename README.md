# dvd logo simulator
a simple recreation of the classic dvd screensaver using html, css and vanilla javascript.

## features
- bouncing logo animation
- random color changes on every bounce
- corner hit counter
- pause and resume with the spacebar

## tech
- html
- css
- javascript

## to run locally
clone the repository and open `index.html` in your browser

```bash
git clone https://github.com/your-username/dvd-logo-simulator.git
```
